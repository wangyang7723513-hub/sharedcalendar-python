# 共享日历 · 团队工作安排

这是一个专为团队设计的共享日历应用，支持 PC 与手机双端自适应访问，界面美观（Apple 极致简约风格）。基于 Python (Flask) 和飞书多维表格（Bitable）实现数据存储与同步。

## ✨ 功能特点

*   **极简操作**：简洁的注册登录流程，基于团队成员的昵称和自定义头像颜色。
*   **优雅设计**：Apple 设计规范，包含毛玻璃效果、平滑动画、轻量级阴影和精致的排版。
*   **工作日历**：清晰的月视图，直观展示每天的工作安排。根据日程密度，带有颜色冷暖变化（热力标识）。
*   **活动热力图**：专属的年度热力图视图（类似 GitHub 贡献图），宏观掌握团队年度活跃与工作排期。
*   **团队视图**：一目了然的团队成员列表，查看协作者信息。
*   **云端协同**：后端基于飞书开放平台多维表格 API 进行数据存储，无需额外部署各种复杂的数据库即可实现数据多端实时同步。

## 🚀 快速开始

本项目由分为前端页面与 Python 后端服务两部分组成。

### 1. 环境准备

确保您的电脑已经安装了 `Python 3.8+`。安装依赖项：

```bash
pip install -r requirements.txt
```

### 2. 飞书多维表格初始化

若您需要将数据存储到自己的飞书多维表格中：
1. 访问飞书开放平台创建应用，并获取最新的 `APP_ID` 和 `APP_SECRET`。
2. 创建两张飞书多维表格，分别用于存储“用户”与“日程”。
3. 在 `server.py` 与 `setup_tables.py` 中替换为您自己的凭证信息。
4. 运行建表初始化脚本：

```bash
python setup_tables.py
```
*(注意：目前代码里已配置了测试用的飞书应用凭证。如果是直接体验，可不用修改配置。)*

### 3. 运行服务

1. 启动 Python 后端服务：

```bash
python server.py
```
2. 服务将在本地启动，访问链接：[http://localhost:5000](http://localhost:5000)
3. 团队成员只需要访问您的 IP 地址（例如：`http://192.168.x.x:5000`）即可共同使用此日历。

## 🛠 技术栈

*   **前端**：HTML5 / JavaScript (ES6+) / Tailwind CSS (通过 CDN 引入)
*   **后端**：Python (Flask)
*   **数据库**：飞书多维表格 Bitable (通过 Open API 调用)
